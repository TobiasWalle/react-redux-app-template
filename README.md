## Installation

First, install [Yeoman](http://yeoman.io) and generator-trb using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g generator-trb
```

Then generate your new project:

```bash
yo trb
```
